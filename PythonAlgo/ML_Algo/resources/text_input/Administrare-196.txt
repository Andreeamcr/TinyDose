Efitard se injecteaza strict intramuscular, cat mai profund (se controleaza, aspirand cu seringa, daca acul nu a patruns intr-un vas sanguin). Posologia este in functie de boala si de bolnav si de asemenea intervalele. Dozele uzuale sunt: Adulti: 400000 UI-800000 UI la 12-14 ore. Copii peste 2 1/2 ani: 200000 UI - 400000 UI la 12-14 ore. In tratamentul sifilisului se recomanda administrarea unei doze totale de 6000000 UI-12000000 UI in decurs de 10-15 zile
